In February 2017, I started working at a place that used Marionette.js (a backbone.js extension) to create a single page application. This was my first experience developing an SPA, so I wrote my own SPA library/toolkit, which is what this site is based on.

Github Page

The example below is code used by this page to load the lite.js code from github and present it. When .attach() is called, it does a HTTP GET request against the contentUrl, inserts the results into the container, then executes the onContentBound event, which calls the third-party 'prism' utility to do the code highlighting.

example usage:


    let code_block = lite.extend({
        container : document.getElementById('lite-code-block'),
        contentUrl : `https://raw.githubusercontent.com/unstableconfiguration/Lite/master/lite.js`,               
        onContentBound : function() {
            require(['prism'], ()=>{ Prism.highlightElement(this.container); });
        }
    });
    new code_block().attach();
